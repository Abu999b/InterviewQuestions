1. What is a computer network?
A computer network is a system of interconnected devices that can communicate with each other to share resources, such as files, printers, and internet connections. Networks can be classified based on their size, such as LAN, WAN, or MAN.
2. What are the types of computer networks?

LAN (Local Area Network): A network confined to a small geographical area, such as a home, office, or campus.

WAN (Wide Area Network): A network that covers a large geographical area, often across cities or countries (e.g., the internet).

MAN (Metropolitan Area Network): A network that spans a city or a large campus.

PAN (Personal Area Network): A small network typically within a range of a few meters, often used for connecting personal devices (e.g., Bluetooth).

4. What is the TCP/IP model?
The TCP/IP model is a four-layer protocol stack used for networking and internet communication. The layers are:
  • Layer 1: Network Interface Layer (equivalent to OSI's Physical and Data Link layers)
  • Layer 2: Internet Layer (equivalent to OSI's Network layer)
  • Layer 3: Transport Layer (equivalent to OSI's Transport layer)
  • Layer 4: Application Layer (equivalent to OSI's Session, Presentation, and Application layers)
5. What is the difference between a hub, switch, and router?

Hub: A basic networking device that broadcasts data to all devices connected to it, leading to inefficiency in large networks.

Switch: A more advanced device that forwards data to specific devices based on MAC addresses, making it more efficient than a hub.

Router: A device that routes data between different networks, often used to connect LANs to the internet, using IP addresses.

6. What is an IP address?
An IP address (Internet Protocol address) is a unique identifier assigned to devices on a network, enabling communication between them. There are two types:
  • IPv4: A 32-bit address (e.g., 192.168.1.1).
  • IPv6: A 128-bit address (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334).
7. What is subnetting?
Subnetting is the process of dividing a larger network into smaller sub-networks (subnets) to improve performance, security, and resource management. It involves splitting the IP address space into smaller blocks.
8. What is a DNS (Domain Name System)?
The DNS is a hierarchical system that translates human-readable domain names (e.g., www.google.com) into IP addresses. It helps users access websites using easily memorable names instead of numeric IP addresses.
9. What is a MAC address?
A MAC (Media Access Control) address is a unique identifier assigned to network interfaces for communication within the data link layer. It is typically burned into the network hardware (e.g., Ethernet or Wi-Fi adapter).
10. What is the difference between TCP and UDP?

TCP (Transmission Control Protocol): A connection-oriented protocol that ensures reliable, error-free communication between devices, using acknowledgments and retransmissions.

UDP (User Datagram Protocol): A connectionless protocol that sends data without acknowledgment, making it faster but less reliable than TCP.

11. What is the function of the transport layer?
The transport layer ensures reliable communication between devices by providing error correction, flow control, and data segmentation. Protocols like TCP and UDP operate at this layer.
12. What is the function of the network layer?
The network layer is responsible for routing data packets from the source to the destination across different networks. It uses logical addressing (IP addresses) and handles packet forwarding and routing.
13. What is the difference between a unicast, multicast, and broadcast?

Unicast: Data is sent from one sender to one specific receiver.

Multicast: Data is sent from one sender to a group of receivers.

Broadcast: Data is sent from one sender to all devices on the network.

14. What is a firewall?
A firewall is a security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules. It can block or allow traffic based on IP addresses, ports, or protocols.
15. What is a VPN (Virtual Private Network)?
A VPN creates a secure, encrypted connection over a public network (e.g., the internet), allowing users to send and receive data as if they were directly connected to a private network.
16. What is the difference between a public and private IP address?

Public IP Address: An address that is accessible over the internet and unique across the globe.

Private IP Address: An address used within a private network, not directly accessible over the internet (e.g., 192.168.x.x).

17. What is NAT (Network Address Translation)?
NAT is a technique used by routers to translate private IP addresses to public IP addresses and vice versa, allowing multiple devices within a private network to share a single public IP address.
18. What is a socket?
A socket is an endpoint for communication between two machines over a network. It provides a programming interface to establish connections using TCP or UDP protocols.
19. What is a routing protocol?
Routing protocols determine the best path for data to travel across networks. Common routing protocols include:
  • RIP (Routing Information Protocol)
  • OSPF (Open Shortest Path First)
  • BGP (Border Gateway Protocol)
20. What is ARP (Address Resolution Protocol)?
ARP is used to map an IP address to a MAC address in a local network. It helps devices communicate with each other on the data link layer.
21. What is a ping command?
The ping command is used to check the connectivity between two devices on a network by sending ICMP echo requests and measuring the response time.
22. What is a VLAN (Virtual Local Area Network)?
A VLAN is a logical grouping of devices on a network that are segmented from other devices, regardless of their physical location, improving network security and performance.
23. What is a packet?
A packet is a unit of data that is transmitted over a network. It contains both the data being sent and control information, such as the destination address and sequence numbers.
24. What is the difference between IPv4 and IPv6?

IPv4: A 32-bit addressing system, providing approximately 4.3 billion unique addresses.

IPv6: A 128-bit addressing system, providing a vastly larger address space (approximately 340 undecillion addresses).

25. What is the OSI Layer 3?
Layer 3 of the OSI model is the Network Layer, responsible for routing packets between devices on different networks. It uses logical addressing (e.g., IP addresses) to ensure data reaches its destination across multiple networks.
26. What is a checksum?
A checksum is a value calculated from a data set, used to verify the integrity of the data during transmission. It ensures that the data received is the same as the data sent by checking for errors in the data.
27. What is the difference between circuit-switched and packet-switched networks?

Circuit-switched Network: A dedicated communication path is established between the sender and receiver for the entire duration of the communication (e.g., traditional telephone networks).

Packet-switched Network: Data is divided into packets, and each packet is routed independently through the network (e.g., the internet). This is more efficient, as it allows for sharing network resources.

28. What is the purpose of a data link layer?
The data link layer ensures the reliable transfer of data over the physical layer. It handles error detection, flow control, and frame synchronization. It is responsible for the logical link between devices on the same network.
29. What is a buffer in networking?
A buffer is a temporary storage area used to store data that is being transferred between devices or networks. It helps in handling differences in speed between the sender and receiver, preventing data loss.
30. What is a collision domain?
A collision domain is a network segment where data packets can collide with each other when two devices transmit simultaneously. In Ethernet networks, hubs create collision domains, while switches limit collision domains to individual ports.
31. What is a broadcast domain?
A broadcast domain is a logical division of a network where all devices can directly receive broadcast messages sent by any device in the domain. Routers separate broadcast domains, while switches can create separate collision domains.
32. What is CSMA/CD?
Carrier Sense Multiple Access with Collision Detection (CSMA/CD) is a network access method used in Ethernet networks to detect and resolve collisions. If a collision occurs, the devices wait for a random period before retrying transmission.
33. What is the role of a switch in a network?
A switch is a networking device that connects devices within a local area network (LAN). It operates at the data link layer and forwards data frames based on MAC addresses, improving network efficiency and reducing collisions compared to hubs.
34. What is DNS resolution?
DNS resolution is the process by which a domain name (like www.example.com) is translated into an IP address (such as 192.168.1.1). This is done through a series of DNS queries and responses, either from a local DNS cache or external DNS servers.
35. What is SSL/TLS in networking?
SSL (Secure Sockets Layer) and TLS (Transport Layer Security) are cryptographic protocols used to secure communications over a network, particularly the internet. They provide encryption, data integrity, and authentication for secure data transmission.
36. What is an HTTP request and response?
An HTTP request is sent by a client (usually a browser) to request resources from a server. The server responds with an HTTP response, which contains the requested resources and status codes (e.g., 200 OK, 404 Not Found).
37. What is FTP (File Transfer Protocol)?
FTP is a protocol used to transfer files between a client and server over a TCP-based network. It uses two channels: one for commands (control connection) and one for data (data connection). FTP can be insecure unless used with encryption (e.g., FTPS).
38. What is the difference between stateful and stateless protocols?

Stateful Protocols: Maintain the state of the connection between the sender and receiver across multiple exchanges (e.g., TCP).

Stateless Protocols: Do not store any session information between requests, each interaction is independent (e.g., HTTP).

39. What is a routing table?
A routing table is a data structure used by routers to determine the best path for forwarding packets to their destination. It contains destination IP addresses, subnet masks, and the next hop information.
40. What is NAT (Network Address Translation)?
NAT is a method used by routers to modify the source or destination IP address in packet headers. It allows multiple devices on a private network to share a single public IP address, helping to conserve IP addresses.
41. What is the role of the session layer in the OSI model?
The session layer is responsible for establishing, managing, and terminating sessions between applications. It ensures that data exchanges occur in an organized and synchronized manner, often providing dialog control (half-duplex or full-duplex).
42. What is the function of the physical layer in networking?
The physical layer is responsible for transmitting raw bitstreams over a physical medium (e.g., cables, radio waves). It defines the electrical, mechanical, and procedural characteristics for data transmission.
43. What is the concept of Quality of Service (QoS)?
Quality of Service (QoS) refers to the management of network resources to ensure that certain types of traffic (e.g., voice or video) receive priority over others (e.g., file downloads). This ensures optimal performance for critical applications.
44. What is a proxy server?
A proxy server acts as an intermediary between a client and a server, handling requests on behalf of the client. It is commonly used for security, privacy, caching, and load balancing purposes.
45. What is the difference between a full-duplex and half-duplex communication?

Full-Duplex: Data can be sent and received simultaneously (e.g., telephones).

Half-Duplex: Data can only be sent in one direction at a time (e.g., walkie-talkies).

46. What is a network topology?
Network topology is the physical or logical arrangement of devices in a network. Common topologies include bus, star, ring, mesh, and hybrid. The topology affects network performance, scalability, and fault tolerance.
47. What is the role of the application layer in networking?
The application layer is the top layer of the OSI model and is responsible for providing network services directly to end-users or applications. Examples of protocols at this layer include HTTP, FTP, and DNS.
48. What is the difference between static and dynamic IP addressing?

Static IP Addressing: The IP address is manually assigned and does not change.

Dynamic IP Addressing: The IP address is automatically assigned by a DHCP server and may change over time.

49. What is the difference between symmetric and asymmetric encryption?

Symmetric Encryption: The same key is used for both encryption and decryption (e.g., AES).

Asymmetric Encryption: Different keys are used for encryption and decryption, typically a public key for encryption and a private key for decryption (e.g., RSA).

50. What is the role of the transport layer in the OSI model?
The transport layer ensures reliable data transfer between hosts. It handles error detection and correction, flow control, and segmentation of large messages into smaller packets. It uses protocols such as TCP and UDP for communication.
